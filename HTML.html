웹 호스팅(github pages)

github

* 장점 : 무료로 업로드한 파일을 이용해 웹사이트를 운영할 수 있도록 컴퓨터와 웹서버를 제공해줌.

  1. 로그인
    • 아이디 : suji95
    • 비밀번호 : 영문4자숫자6자

  2. 저장소 생성
    3. 오른쪽 위 + 버튼 클릭 > New repository 클릭
    Repository name(지정하고자 하는 저장소 이름) / Public 체크확인 / Initialize this repository with a README 체크

    * New repository 의미 : 저장소

  3. 파일 업로드
    4. 1) Upload files 클릭
    2) choose your files(파일 선택하세요) 클릭 후 업로드 할 파일 다중 선택
    3) 변경된 내용 쓰기(파일이 변경될 때마다 업로드 해야하기 때문에 변경 내용 써야함)
    4) Commit changes 버튼 클릭

  4. 업로드한 웹 페이지를 인터넷을 통해 서비스
    5. 1) Setting 버튼 클릭
    2) Github Pages 항목에서 Source 밑 드롭박스에서 master branch를 선택 -> Select branch 중 master branch를 선택 -> save 버튼 클릭
    3) 주소를 제공해줌.
